# 欢迎来到我的个人博客

本站是由基于Python的Mkdocs工具搭建的个人博客，使用Material for MkDocs主题，同时参考了Wcowin大佬的美化格式。

网站以静态网页的形式部署在Github Pages上，可以完全使用Markdown语言编写文章，并通过Material for Mkdocs生成博客样式。
