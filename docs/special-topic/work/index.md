# 参与过的课程设计/竞赛作品汇总


## 课程设计

!!! note "课程设计是什么？"
    课程设计是指在学校课程中开设的设计任务，一般为团队协作的大作业形式，要求学生完成设计方案并提交给老师，老师对设计方案及最终作品进行评审，并给出评分。
