    Ubuntu操作系统中的一些常见配置

    一个崭新的Ubuntu操作系统，往往需要进行一些基本配置，以便可以更高效地使用。

    切换安装源

    Bash
    1
2
3
4
5
    # 备份系统默认源文件
sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak

# 更换为清华源
sudo vim /etc/apt/sources.list

    镜像源地址：https://mirrors.tuna.tsinghua.edu.cn/help/ubuntu/

    找到对应的Ubuntu版本，复制其中的源地址，替换掉原来的源地址。

    Bash
    1
2
    # 更新安装源列表
sudo apt update && sudo apt -y upgrade

    配置Fish终端

    Fish是一款功能强大的命令行shell，可以替代bash。

    Bash
    1
2
3
4
5
6
7
8
    # 安装Fish
sudo apt install fish
# 将Fish设置为默认shell
chsh -s /usr/bin/fish
# Fish初始化文件
sudo vim ~/.config/fish/config.fish
# 设置用户的全局环境变量，在config.fish中添加
set -x VAR VALUE

    # 安装网络工具

# 查看网卡IP地址

sudo apt install ssh
# 启动SSH服务
sudo systemctl start ssh
